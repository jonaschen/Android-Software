# Research Log — 2026-04-07

## Session Type
Periodic AOSP research session (steward agent)

## Topics Researched

1. **Android 16 GKI / Kernel** — Confirmed android16-6.12 branch (Linux 6.12 LTS). New: memory allocation profiling system, BPF toolchain moved to CO-RE. Already captured in HS-023.

2. **AIDL/HIDL HAL Status** — No change from prior research. HIDL deprecated since A11, AIDL mandatory for new HALs. Audio CAP gap persists. Already captured in HS-025.

3. **pKVM / AVF** — Confirmed LL-NDK vendor VMs, early-boot VMs, Microdroid 16K, FF-A guest firmware, Linux Terminal. New: pKVM vendor module registration simplified in A16 (no additional init step). Already captured in HS-024.

4. **SELinux** — **NEW**: Android 16 QPR2 adds SELinux macro to harden kernel driver IOCTLs. Blocks deprecated/dev IOCTLs in production, limits profiling IOCTLs to shell/debuggable. Created HS-027.

5. **16KB Page Size** — **NEW deadline detail**: Mandatory compliance by May 31, 2026 for Play Store. One-time extension available but only to that date. Non-compliant apps blocked from updates. Created HS-028.

6. **Connectivity / Network** — **NEW**: DHCPv6 Prefix Delegation rolling out via Play System Update (A11+). Android 16 classifies IPv6 link-local/multicast as local network traffic gated by new runtime permission. QPR3 splits Adaptive Connectivity into Wi-Fi stability + battery saving. Created HS-029.

7. **Build System (Soong/Bazel)** — **NEW**: Genrules sandboxed (can only access listed srcs), Python 2 fully removed, partition images only include clean-build contents. Bazel migration incremental with Soong plugins needing manual porting. Created HS-030.

8. **Init / Boot Sequence** — No significant A16-specific changes found in public documentation. Early-boot VM support (from AVF) is the main boot-related change, already covered in HS-024.
