## Key Findings

### 1. April 2026 ASB — Two Notable CVEs

**CVE-2026-0049 (Critical, Framework)**
- Zero-interaction local DoS — no user action, no elevated privileges
- Affects Android 14, 15, 16, and 16-QPR2 simultaneously
- Patch level 2026-04-01
- Not known to be exploited in the wild
- Source code patch expected in AOSP within 48 hours of bulletin

**CVE-2025-48651 (High, StrongBox)**
- Hardware secure-element keystore flaw
- Impacts Google, NXP, STMicroelectronics, Thales SE implementations
- Patch level 2026-04-05
- Fix path is vendor-side SE firmware — platform-only updates do NOT close the gap
- AOSP bug IDs: A-434039170, A-467765081, A-467765894, A-467762899

→ **New hindsight note**: `HS-045_april_2026_asb_framework_dos_strongbox.md`
